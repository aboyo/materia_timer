# material_timer

實作 Google Chrome 中 Timer Widget，使用`React.js`開發並採用`Next.js` framework 實作，狀態管理部分使用`Recoil.js`，UI 部分使用了 [Material-UI library](https://material-ui.com/ "Material-UI library")。

## Usage

npm install

```bash
npm install
```

執行 dev 模式

```bash
npm run dev
```

或是

```bash
npm run build
npm start
```

開啟瀏覽器：http://127.0.0.1/timer
